Directory for the API script to search the Internet Archive database and download sound file items.

Please install version 0.9.8 of the internetarchive with the following command.

pip3 install -Iv internetarchive==0.9.8

Documentation on how to use the official python wrapper for the API (i.e. internetarchive 0.9.8) can be found in the following link, https://pypi.python.org/pypi/internetarchive/0.9.8

FILES

API_backend.py: Python script that queries the Internet Archive API and downloads sound file items. Not intended to be used as stand-alone script.
